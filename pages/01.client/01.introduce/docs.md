DataHub Client 是 DataHub 的命令行客户端，用来执行 DataHub 相关命令。

| 命令	    | 用途              |
| :---------- | :----------       |
| dp          | Datapool 管理      |
| subs        | Subscrption 管理   |
| login       | 登录到DataHub Client端 | 
| pull        | 下载数据           |
| pub         | 发布数据           |
| repo        | Repository 管理   |
| job         | 显示任务列表       |
| ep		  | 设置Entrypoint    |
| logout	  | 登出              |
| help        | 帮助命令          |  


** DataHub Client命令行使用说明 **

- 参数支持全名和简称两种形式，例如--type等同于-t。详情见命令帮助。
- 参数赋值支持空格和等号两种形式，例如--type=file等同于--type file。
